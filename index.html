关于 Cashlens · 提示词 v4.0
产品
Cashlens 是给香港中小企业用的银行月结单「统计流水」工具,只算 debit 存入交易,绝不涉及做账/审计/税项,输出可被"做账审计板块"直接读取的 TSV 结构化数据。
数据安全
  • 全部前端运行,无后端服务器
  • API Key 仅保存在本机浏览器 localStorage(无后端中转)
  • 上传文件仅在浏览器内读取,直接通过 HTTPS 发往你选择的模型服务商
  • 关闭网页 = 数据自动清除(localStorage 除外)
v4.0 统计规则
  • 仅算 debit 存入;credit 出账整行跳过
  • debit 中再排除:利息 / 定存 / 内部兑换 / 关联方 / 银行费用 / 信用卡还款 / 股东注资
  • 每笔分类不确定 → 进「待用户确认事项」清单,不允许 AI 猜测
汇率政策
  1. 银行月结单实际汇率(若有)
  2. HKMA 当日中间价
  3. HKMA 月平均中间价
  4. 财政司司长汇率(仅税表)
