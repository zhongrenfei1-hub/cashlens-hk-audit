关于 Cashlens · 提示词 v4.0
产品
Cashlens 是给香港中小企业 / 审计师事务所用的银行月结单流水分析工具。基于 Anthropic Claude API,符合 HKSA 230 工作底稿要求。
数据安全
  • 全部前端运行,无后端服务器
  • API Key 仅保存在本机浏览器 localStorage(无后端中转)
  • 上传文件仅在浏览器内读取,直接通过 HTTPS 发往你选择的模型服务商
  • 关闭网页 = 数据自动清除(localStorage 除外)
v4.0 排除规则
  • 利息 / 定存 / 内部兑换 / 关联方 / 银行费用 / 出账 / 信用卡还款 / 股东注资
  • 每笔 borderline 必须在审计轨迹引用 IRO / DIPN / SME-FRS 条款
汇率政策
  1. 银行月结单实际汇率(若有)
  2. HKMA 当日中间价
  3. HKMA 月平均中间价
  4. 财政司司长汇率(仅税表)
